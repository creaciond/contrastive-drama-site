Data

This web application uses data from the Drama Corpora project. DraCor offers 11 drama corpora (and some other are currently being prepared for public access). All dramas are TEI-encoded.

As well as encoded play texts, DraCor offers a variety of tools to interact with play data: Easy Linavis & Shiny app to discover play networks as well as API (+Swagger interactive documentation) and SPARQL endpoints for custom data requests.

Metrics

Three metrics are available:

Frequency

For lemmas — outputs a word cloud, for parts of speech — a scatter plot of dots (each dot = new play), with a line being a linear approximation.

Key items

Calculated with log-likelihood metric. The corpus is split into two parts, each part is compared against the whole corpus.

Items preferred/items avoided

Based on Craig's Zeta metric firstly implemented in the stylo package, later transferred to Python in pyzeta.

Code

Is available in two repositories:

  • creaciond/contrastive-drama for main code,
  • creaciond/contrastive-drama-site for web application.
